## instalación

1. Instala [git](https://git-scm.com/book/es/v2/Inicio---Sobre-el-Control-de-Versiones-Instalaci%C3%B3n-de-Git).
2. Crear un directorio donde clonar el respositorio (ejemplo/documents/codevs).
3. Clona el repositorio `git clone https://github.com/CoDevsCommunity/codevs-web-backend.git ` .
4. Ubicate en el directorio `cd codevs-web-backend`.
5. Instala las dependencias `npm install`
6. Inicie la aplicacion `npm start`

## tecnologías de desarrollo

- **Node.JS** Es un entorno JavaScript que nos permite ejecutar en el servidor, de manera asíncrona, con una arquitectura orientada a eventos y basado en el motor V8 de Google. Es una plataforma que avanza muy rápidamente y cada vez está más presente en el mercado.
